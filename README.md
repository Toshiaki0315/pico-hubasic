# Pico-HuBASIC

Pico-HuBASICは、Raspberry Pi Pico 2 (RP2350) と Waveshare RP2350-Touch-LCD-2.8 向けに開発されている、1980年代のマイコン（Hu-BASIC等）ライクな BASICインタプリタです。

現在、構文解析エンジンや制御構造（IF文、FOR文等）のコア機能の実装が進んでおり、将来的にはSDカードへのプログラム保存やLCD画面へのグラフィック描画機能も備える予定です。

---

## 🚀 Picoへの環境構築と転送方法（ビルド）

実機の Pico 2 (RP2350) を動かすための手順です。

### 1. 必要な環境ソフト
* **Raspberry Pi Pico SDK (v2.0.0以上)**: RP2350をビルドするためにPico SDKの最新バージョンが必要です。
* **CMake および C/C++ コンパイラ (GCC/Clang)**: クロスコンパイル環境 (arm-none-eabi-gcc等)

### 2. ビルド手順
ターミナルを開き、Pico-HuBASICのディレクトリで以下のコマンドを実行します。

```bash
mkdir build
cd build
# 通常のPico向けビルド
cmake ..
make
```

> **Note (PC・ホスト環境でのテストビルドについて):**
> 開発者向けとして、実機がなくてもPC上（Mac / Linux 等）で動作検証できるテスト環境を用意しています。  
> `cmake -DBUILD_TESTS=ON ..` とすることで、Pico SDK を使わずに Google Test を利用したコアエンジンの単体テストをビルドできます。

### 3. Pico実機への転送（書き込み）
1. Raspberry Pi Pico のボード上にある **BOOTSELボタン** を押しながら、PCにUSBケーブルで接続します。
2. PCに `RPI-RP2` という名前のUSBメモリ（ドライブ）として認識されます。
3. 手順2でビルドして生成された `build/pico_hubasic.uf2` というファイルを、その `RPI-RP2` ドライブにドラッグ＆ドロップして書き込みます。
4. コピーが完了するとPicoが自動的に再起動し、Pico-HuBASICが起動します。

### 4. ターミナルでの接続
Pico は USB CDC（シリアル接続）として認識されます。
* **Windows**: TeraTerm や PuTTY を使い、認識されたシリアルポート（COMポート）へ接続します。
* **Mac/Linux**: ターミナルから `screen /dev/tty.usbmodem* 115200` や `minicom` を使って接続します。

## 📖 Pico-HuBASIC 操作マニュアル（初心者向け）

ターミナルに接続すると、`Pico-HuBASIC v2.0` と `Ready` という文字が表示され、コマンドやプログラムを入力することができます。

### 1. 基本的な使い方（ダイレクトモード）
行番号（数字）を付けずにコマンドを入力すると、Enterキーを押した瞬間にそのまま実行されます。これをダイレクトモードと呼びます。

**画面に文字や計算結果を出す (`PRINT`)**
```basic
PRINT 10 + 20
30
```

**変数に数字を入れる (`LET`)**
変数（アルファベット）の中に数字を記憶させることができます。`LET` は省略することもできます。
```basic
A = 10
B = 5
PRINT A * B
50
```

**使える四則演算・記号**
* `+` (足し算), `-` (引き算)
* `*` (掛け算), `/` (割り算)
* `(` `)` (カッコ：計算の優先順位を変えます。例: `(1 + 2) * 3`)

### 2. プログラムを作る（現在開発中）
行番号（数字）を最初につけてコマンドを入力すると、すぐには実行されずメモリに記録されます。これを組み合わせることで複雑なプログラムを作ることができます。※現在のバージョンではREPL経由での行登録は試験的です。

```basic
10 A = 1
20 PRINT A
30 A = A + 1
40 IF A <= 5 THEN GOTO 20
```
*(上記を入力した後、後日実装予定の `RUN` コマンドを入れることで、上から下へと自動で実行されます)*

### 3. ステートメント（命令）辞典

現在実機（テストエンジン）に搭載されている主な命令です。

#### `GOTO`（指定した行へジャンプ）
プログラムの実行順序を、指定した行番号に強制的に移動させます。
```basic
10 PRINT 1
20 GOTO 40
30 PRINT 2   <- この行は飛ばされるため表示されません
40 PRINT 3
```

#### `IF ~ THEN`（条件分岐）
「もし〜なら、〜する」という条件を作ります。
条件に使える記号は `=`, `>`, `<`, `>=`, `<=`, `<>`（等しくない）です。
```basic
IF A > 10 THEN PRINT A
```
条件が正しいときだけ、`THEN` のあとの命令が実行されます。間違っているときはその行は無視されます。

#### `FOR ~ NEXT`（繰り返しループ）
決められた回数だけ、命令をくり返したいときに使います。
```basic
10 FOR I = 1 TO 3
20 PRINT I
30 NEXT I
```
実行すると `1` `2` `3` と順番に表示されます。
`TO` のあとに `STEP 増える数` を加えることで、一度に増える数を買えたり、マイナスして引き算していくことも可能です（例: `FOR I = 10 TO 1 STEP -1`）。
