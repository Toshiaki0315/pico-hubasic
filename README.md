# pico-basic

**Raspberry Pi Pico 2 (RP2350) 上で動作する、X1 turbo (CZ-8FB02) 互換のネイティブ Hu-BASIC インタプリタ** です。

1980年代のマイコンの熱気を、現代のパワフルなマイクロコントローラで蘇らせることを目的としています。コア言語・グラフィック・サウンド・ファイル API などは `parser` と HAL で実装が進んでおり、Pico 実機では MicroSD（FatFS）統合や一部命令の完成度を高める作業が続いています。

命令の実装状況の一覧は [docs/COMMAND_STATUS.md](./docs/COMMAND_STATUS.md) を参照してください。ユーザ向けの詳細な説明は [MANUAL.md](./MANUAL.md) です。

## 🎯 ターゲットハードウェア

本プロジェクトは、以下のハードウェア環境をメインターゲットとして開発を行っています。
* **Raspberry Pi Pico 2** (RP2350)
* **[Waveshare RP2350-Touch-LCD-2.8](https://www.waveshare.com/wiki/RP2350-Touch-LCD-2.8)** (2.8インチ SPI LCD / タッチパネル / MicroSDスロット内蔵ボード)

## 📖 pico-basic 操作ガイド（初心者向け）

ターミナルに接続すると、`pico-basic` の起動メッセージと `Ready` という文字が表示され、コマンドやプログラムを入力することができます。

### 1. 基本的な使い方（ダイレクトモード）
行番号（数字）を付けずにコマンドを入力すると、Enterキーを押した瞬間にそのまま実行されます。これをダイレクトモードと呼びます。

**画面に文字や計算結果を出す (`PRINT`)**
```basic
PRINT 10 + 20
30
```

**変数に数字を入れる (`LET`)**
変数（アルファベット）の中に数字を記憶させることができます。`LET` は省略することもできます。
```basic
A = 10
B = 5
PRINT A * B
50
```

**使える四則演算・記号**
* `+` (足し算), `-` (引き算)
* `*` (掛け算), `/` (割り算)
* `(` `)` (カッコ：計算の優先順位を変えます。例: `(1 + 2) * 3`)

### 2. プログラムを作る（行番号モード）
行番号（数字）を行頭につけると、その行はすぐには実行されずプログラム領域に格納されます。行番号のない行（ダイレクトモード）で **`LIST`** と入力すると一覧表示、**`RUN`** で先頭行から実行、**`NEW`**（または **`CLEAR`**）でプログラムと変数を消去できます。

```basic
10 A = 1
20 PRINT A
30 A = A + 1
40 IF A <= 5 THEN GOTO 20
RUN
```

### 3. ステートメント（命令）辞典

現在実機（テストエンジン）に搭載されている主な命令です。

#### `GOTO`（指定した行へジャンプ）
プログラムの実行順序を、指定した行番号に強制的に移動させます。
```basic
10 PRINT 1
20 GOTO 40
30 PRINT 2   <- この行は飛ばされるため表示されません
40 PRINT 3
```

#### `IF ~ THEN`（条件分岐）
「もし〜なら、〜する」という条件を作ります。
条件に使える記号は `=`, `>`, `<`, `>=`, `<=`, `<>`（等しくない）です。
```basic
IF A > 10 THEN PRINT A
```
条件が正しいときだけ、`THEN` のあとの命令が実行されます。間違っているときはその行は無視されます。

#### `FOR ~ NEXT`（繰り返しループ）
決められた回数だけ、命令をくり返したいときに使います。
```basic
10 FOR I = 1 TO 3
20 PRINT I
30 NEXT I
```
実行すると `1` `2` `3` と順番に表示されます。
`TO` のあとに `STEP 増える数` を加えることで、一度に増える数を変えたり、マイナスして引き算していくことも可能です（例: `FOR I = 10 TO 1 STEP -1`）。
