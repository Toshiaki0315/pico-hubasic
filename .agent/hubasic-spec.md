# Specification: Hu-BASIC Compatibility Layer

このファイルは、pico-hubasic が準拠すべき Hu-BASIC の言語仕様と内部構造を定義します。

## 1. 言語仕様の概要
- **基本互換性**: SHARP MZシリーズ等に搭載されていた Hu-BASIC のコマンド体系をターゲットとする。
- **行番号**: 1 ～ 65535。プログラムは常に行番号の昇順で保持される。
- **変数名**: 英字1文字、または英字+数字の2文字（例: `A`, `B1`）。
    - 実数型: `A` (デフォルト)
    - 文字列型: `A$`
    - 整数型: `A%`
- **配列**: `DIM` 命令により定義。多次元配列をサポート。

## 2. メモリマップ (Pico 2 割当案)
エージェントはこのアドレス空間を意識してポインタ操作やメモリ管理コードを生成すること。
- `0x0000 - 0x7FFF`: BASIC テキスト領域 (32KB - 拡張可能)
- `0x8000 - 0x9FFF`: 変数・配列格納領域
- `0xA000 - 0xBFFF`: 文字列バッファ領域
- `0xC000 - 0xDFFF`: VRAM領域 (320x240, 8bpp 等の構成を検討)
- `0xE000 - 0xFFFF`: システムワークエリア・スタック

## 3. 中間言語 (Token) 定義
プログラム保存時のトークン化ルールです。
- `0x80`: `LIST`
- `0x81`: `RUN`
- `0x82`: `PRINT` / `?`
- `0x83`: `IF`
- `0x84`: `THEN`
- `0x85`: `FOR`
- `0x86`: `NEXT`
- `0x87`: `GOTO`
- `0x88`: `GOSUB`
- `0x89`: `RETURN`
*(※実装が進むにつれ、実際のトークン表をここに追記していく)*

## 4. 特徴的な挙動・コマンド
Hu-BASIC 特有の仕様をここに記述します。
- **`INIT`**: メモリや変数の初期化。
- **`COLOR`**: 前景色、背景色の指定。Hu-BASIC 独特のカラーコード（0-7等）を再現する。
- **`GET`**: キー入力を1文字取得（待機しない）。
- **`CURSOR`**: カーソル位置の指定（`LOCATE` 相当）。
- **数式評価**: 演算優先順位は、括弧 > 関数 > 巾乗 > 乗除 > 加減 > 比較 > 論理演算。

## 5. 拡張仕様 (Pico 2 独自)
- **`GPIO`**: `GPIO PIN, MODE, VALUE` のような形式で Pico 2 のピンを直接制御。
- **`LCD`**: 2.8インチ液晶へのグラフィック描画命令 (`PSET`, `LINE`, `CIRCLE`, `FILL`)。
- **`FILES`**: SDカード内のファイル一覧表示。
