# 実機手動テストチェックリスト（Waveshare RP2350-Touch-LCD-2.8 想定）

ホスト単体テストではカバーできない、USB・液晶・SD・タッチ・音声などの確認用です。リリース前やハード変更後に実施することを推奨します。

**準備:** `pico_basic.uf2` を書き込み済みのボード、データ対応 USB ケーブル、シリアルターミナル（VS Code Pico Serial Monitor や `screen` / `minicom` 等）。

---

## A. 起動・シリアル（USB CDC）

| # | 確認内容 | 手順 | 期待結果 | 合否 |
|:-:|:---|:---|:---|:---:|
| A1 | 起動メッセージ | 電源投入または USB 接続後、シリアルを開く | `pico-basic v2.0` および `Ready` が表示される | [ ] |
| A2 | エコーバック | `PRINT 2+3` を入力 | `5` と改行後に再び `Ready` | [ ] |

| A4 | リスト | `10 PRINT "OK"` 登録後 `LIST` | 登録行が表示される | [ ] |
